# Data-Validation
Column B (Store_Code): Apply Text Length validation so the code is exactly 3 characters.

Column C (Region): Apply a List validation using: APAC, EMEA, NA, LATAM.

Column D (Sale_Date): Apply Date validation for the 2018 calendar year only.

Column E (Units_Sold): Apply Whole Number validation for values between 0 and 5000.

Column F (Username): Apply Custom validation to ensure there are no spaces in the name.
